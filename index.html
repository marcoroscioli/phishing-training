What is Email Phishing?

Email phishing is a cybercrime where attackers send fraudulent/suspicious emails designed to trick recipients into revealing sensitive information, downloading malicious software, or taking actions that compromise security.

üö® Common Phishing Techniques

1. Urgency and Fear Tactics

Red Flags:
  • "Your account will be suspended in 24 hours!"
  • "Immediate action required"
  • "Security breach detected"
  • "Click now or lose access"

2. Impersonation Attacks

Red Flags:
  • Emails from "IT Support," "HR Department," or "CEO"
  • Slight misspellings in sender domains
  • Generic greetings like "Dear Employee"

3. Credential Harvesting

Example: A fake Microsoft 365 login page designed to steal your work credentials.

4. Malicious Attachments

Warning Signs:
  • Unexpected .exe, .zip, .scr files
  • "Invoice" or "Receipt" from unknown senders
  • Requests to "enable macros"

üîç How to Identify Phishing Emails

Security Checklist:

  • Check sender's email address carefully
  • Look for poor grammar and spelling
  • Verify unexpected requests independently
  • Hover over links without clicking
  • Be cautious of urgent language
  • Question generic greetings

üõ°Ô∏è Best Practices for Protection

  • Pause and think before clicking or responding
  • Verify independently through different communication channels
  • Use official websites instead of email links
  • Keep software updated with latest security patches
  • Report suspicious emails to IT security immediately

üö® If You Suspect Phishing

Immediate Actions:
1. Don't click, download, or respond
2. Take a screenshot for documentation
3. Report to IT security following company procedures
4. If you've already clicked - change passwords immediately and notify IT
